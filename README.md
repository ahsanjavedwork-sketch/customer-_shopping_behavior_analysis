# Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from data loading and exploration to dashboard creation and reporting. The goal is to extract meaningful insights from raw data using Python, SQL, and Power BI.

---

## Dataset

The dataset used in this project contains structured business data that was analyzed to identify trends, patterns, and key performance indicators (KPIs).

**Dataset Features:**

* Multiple data attributes and records
* Numerical and categorical variables
* Real-world business insights

## Project Steps

### 1. Data Loading

* Imported dataset into Python.
* Verified data structure and data types.

### 2. Exploratory Data Analysis (EDA)

* Examined dataset dimensions.
* Generated summary statistics.
* Identified trends, distributions, and correlations.
* Visualized key metrics using charts and graphs.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data entries.
* Standardized column formats.

### 4. SQL Analysis

* Imported cleaned data into the database.
* Executed SQL queries for:

  * Data filtering
  * Aggregations
  * Joins
  * Grouping and sorting
  * KPI calculations

### 5. Dashboard Development

* Connected Power BI to the processed dataset.
* Created interactive dashboards.
* Added filters, slicers, and visualizations.
* Monitored business KPIs and trends.

### 6. Reporting

* Summarized findings and business insights.
* Documented methodology and results.
* Prepared stakeholder-friendly reports.

### 7. Presentation Creation

* Designed a professional presentation using Gamma.
* Highlighted objectives, methodology, dashboard screenshots, and key insights.

## How to Run

### Prerequisites

* Python 3.x
* PostgreSQL/MySQL/SQL Server
* Power BI Desktop

### Installation

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Analysis

```bash
python analysis.py
```

### Execute SQL Queries

1. Import the cleaned dataset into your database.
2. Run the SQL scripts located in the `SQL/` folder.

### Open Dashboard

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh data connections if required.
